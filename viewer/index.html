Technical Details

Demonstrates how to embedd a functional machine learning model and application logic directly into a standard PNG image.

Steganography

Data is stored in the RGB channels of the image pixels. Alpha is kept at 255 to maintain visual consistency. The viewer uses a custom JS routine to reconstruct the binary blob from the Canvas ImageData buffer.

Machine Learning

The intent classifier is a Logistic Regression model trained on email snippets. It doesn't use heavy libraries; it's implemented in pure Python (training) and pure JavaScript (inference).

Quantization

Weights are quantized to int8 to reduce the payload size. The total "brain" (model + code + UI) is only 3 KB, allowing it to fit into a tiny ~100x100px area of the image.
